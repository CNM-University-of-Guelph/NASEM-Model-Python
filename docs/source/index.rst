================================
NASEM Dairy Python Documentation
================================

This package is a Python implementation of the Nutrient Requirments of Dairy Cattle 8th revisied edition (NASEM, 2021) model.
The textbook detailing the model is available `here <https://nap.nationalacademies.org/catalog/25806/nutrient-requirements-of-dairy-cattle-eighth-revised-edition>`_.
